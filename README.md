# WhatsApp Chatbot Backend Simulation

## 📌 Description
This project simulates a WhatsApp chatbot backend using Java and Spring Boot. It provides a REST API endpoint to receive messages and respond with predefined replies.

---

## 🚀 Features
- REST API endpoint `/webhook`
- Accepts JSON input
- Responds with predefined messages
- Logs incoming messages

---

## 🛠 Tech Stack
- Java
- Spring Boot
- Maven

---

## 📡 API Endpoint

### POST /webhook

### Request:
```json
{
  "message": "Hi"
}
